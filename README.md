# Caderno Temático com IA: Metodologias Ágeis utilizando NotebookLM

## Descrição do Projeto
Este projeto foi desenvolvido como atividade prática da DIO com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa, organização do conhecimento e engenharia de prompts.
O projeto utiliza o NotebookLM para estudar metodologias ágeis, sintetizar conteúdos, gerar resumos estruturados, criar perguntas de revisão e documentar o processo de interação com IA.

---

# Objetivos
- Estudar os principais conceitos de metodologias ágeis;
- Utilizar IA como apoio ao aprendizado técnico;
- Desenvolver habilidades de engenharia de prompts;
- Organizar conteúdos técnicos de forma estruturada;
- Criar um material reutilizável para futuras revisões e aplicações práticas.

# Resumos Estruturados

## Scrum

Framework ágil utilizado para gerenciamento iterativo e incremental de projetos.

### Principais Características

- Trabalho em Sprints;
- Reuniões diárias;
- Backlog priorizado;
- Entregas contínuas;
- Feedback constante.

---

## Kanban

Método visual de gerenciamento de fluxo de trabalho.

### Principais Características

- Fluxo contínuo;
- Organização visual;
- Limites de tarefas em andamento;
- Flexibilidade;
- Melhoria contínua.

---

## Lean

Metodologia focada na redução de desperdícios e otimização de processos.

### Objetivos

- aumentar eficiência;
- reduzir desperdícios;
- melhorar fluxo de trabalho;
- gerar mais valor ao usuário.

# Perguntas e Respostas

## O que é Scrum?

Framework ágil baseado em entregas incrementais realizadas em ciclos curtos.

---

## O que é Kanban?

Método visual utilizado para gerenciamento contínuo de tarefas.

---

## Quando usar Scrum?

Em projetos complexos que exigem planejamento iterativo e colaboração constante.

---

## Quando usar Kanban?

Em fluxos contínuos de trabalho e demandas recorrentes.
